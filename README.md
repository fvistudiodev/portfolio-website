# Personal Portfolio

This repository contains the source code for my personal portfolio website.

The website highlights my projects, skills, and experience in web development.

## Tech Stack

* HTML5
* CSS3
* JavaScript

## Run Locally

Simply open `index.html` in your browser, or use a local development server.

For example, with the VS Code Live Server extension:

```bash
Right Click → Open with Live Server
```
